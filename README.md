# 🎬 Netflix Clone

A responsive **Netflix Clone** built using **HTML** and **CSS**. This project recreates the look and feel of the Netflix landing page with a modern, responsive design. It was created as a personal project to improve my frontend web development skills.

---

## 📖 Project Overview

The Netflix Clone is a static frontend website inspired by the official Netflix homepage. It focuses on creating a clean user interface, responsive layouts, and attractive styling using only HTML and CSS.

---
